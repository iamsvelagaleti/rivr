    A AudioFile represents an audio file with an optional synthesis text backup.
    AudioItem
    This abstract class is the superclass of all classes representing an audio element in the dialogue.
    Mark
    A Mark represents a marker in an AudioItem sequence which can be used to determine when a user started to speak.
    Pause
    A Pause represents a silence of a specified duration in an AudioItem sequence.
    SpeechSynthesis
    A SpeechSynthesis represents a text that will be spoken using a synthesis voice.
