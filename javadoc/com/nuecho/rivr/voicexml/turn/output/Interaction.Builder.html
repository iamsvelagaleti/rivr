Class Interaction.Builder

  • java.lang.Object
    • com.nuecho.rivr.voicexml.turn.output.Interaction.Builder
  • Enclosing class:
    Interaction

    public static final class Interaction.Builder
extends Object
    Builder used to ease the creation of instances of Interaction.

    Building an Interaction implies the following steps:

    • Add some prompts
      • For each prompt, speech/DTMF recognition can be specified (thus enabling barge-in when the prompt is played).
    • Once all prompts are added, optionally specify either:
      • a final recognition window (speech or DTMF)
      • a final recording window

    At any time, it is possible to change the current language used for prompts (relevant to speech synthesis) and the barge-in type, i.e. speech or hotword.

    This can be translated to: 

     Builder builder = new Builder();
 builder.addPrompt(...);
 builder.addPrompt(...);
 //... repeat as needed
 Interaction interaction = builder.build(...);

      • setBargeInType

        public Interaction.Builder setBargeInType(Interaction.BargeInType bargeInType)
        Sets the barge-in type to either Interaction.BargeInType.speech or Interaction.BargeInType.hotword for the prompts that will be added using one of the addPrompt(...) methods.

        Note: When a Interaction.Builder is created, the default value for this flag is null, meaning the barge-in type will be platform-dependant.

        Parameters:
        bargeInType -
        • Interaction.BargeInType.speech
        • Interaction.BargeInType.hotword
        • null reverts to platform default value
        Returns:
        this builder

      • setLanguage

        public Interaction.Builder setLanguage(String language)
        Sets the language code for the prompts that will be added using one of the addPrompt(...) methods.

        Note: When a @{link Builder} is created, the default value for this property is null, i.e. no explicit language code will be generated in the VoiceXML thus relying on the platform-specific default language code.

        Parameters:
        language - language code for this prompt. null if language should be reset to platform-specific default value for the prompts to be added.
        Returns:
        this builder
